# Local Dev Script Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a single local management script that can start, stop, restart, inspect, and tail the BiliNote frontend and backend without Docker.

**Architecture:** Add one repo-level shell script that owns PID files, log files, and launch commands for both services. Keep behavior configurable through environment overrides so tests can run against fake commands instead of real servers.

**Tech Stack:** POSIX shell/Bash, `uv`, `pnpm`, Python `unittest`
